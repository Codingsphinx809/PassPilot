# PassPilot

PassPilot is a modern certification learning platform designed to help professionals prepare for industry certifications through personalized, engaging, and data-driven learning.

## Current Focus

The first certification supported by PassPilot is:

- ITIL Foundation V5

The platform is designed to expand to additional certifications, including:

- AWS
- Azure
- PMP
- PRINCE2
- Scrum
- Security+
- Network+
- CCNA
- ISO 27001

## Technology

- React Native
- Expo
- TypeScript
- Supabase
- PostgreSQL
- GitHub

## Vision

We're not building another quiz app.

We're building the platform professionals trust to know when they're ready to pass.
