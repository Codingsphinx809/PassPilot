# PassPilot Database

## Core Tables

### Profiles

Stores user information.

### Courses

Stores all certifications offered by PassPilot.

Examples:

- ITIL Foundation V5
- AWS Cloud Practitioner
- PMP
- Scrum

### Modules

Groups lessons into logical sections.

### Lessons

Contains learning content.

### Quizzes

Question banks linked to lessons.

### Flashcards

Review cards using spaced repetition.

### Study Plans

Personalized schedules for learners.

### Progress

Tracks lesson completion.

### Exam Attempts

Stores mock exam history.

### Achievements

Badges, streaks and rewards.

## Principles

- Normalized schema
- Secure by default
- Scalable
- Multi-certification support
