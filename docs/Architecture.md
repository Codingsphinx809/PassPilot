# PassPilot Architecture

## Overview

PassPilot is built as a modular certification learning platform.

The first application is the mobile app, but the architecture is designed to support multiple applications.

## Technology Stack

### Mobile

- React Native
- Expo
- Expo Router
- TypeScript

### Backend

- Supabase
- PostgreSQL
- Row Level Security
- Authentication

### Version Control

- Git
- GitHub

## Design Principles

- Feature-based architecture
- Reusable components
- Strong typing
- Offline-first where practical
- Shared business logic
- Scalable to multiple certifications

## Planned Applications

- Mobile App
- Marketing Website
- Admin Portal

## Future

The platform will eventually become a monorepo with shared packages and applications.
